PCT-100-003

化工控制实训装置

基于 ESP32-C3 的智能化工过程控制实训平台，集成 WiFi 远程监控、MQTT 物联上报与传感器闭环控制
ESP32-C3 WiFi 远程 MQTT 物联 光感控制 温度监测

产品概述

PCT-100-003 是一款面向化工类职业院校与实训基地的智能过程控制训练设备。以 ESP32-C3 为主控核心，集成环境感知、逻辑控制、物联网通信三大模块，帮助学生理解化工生产中的自动检测、闭环调节与远程监控原理。

核心特性

📡

WiFi 远程控制

板载 Web 服务器，同一局域网下手机/电脑浏览器即可访问控制面板，无需安装任何软件。

🌐

MQTT 物联上报

支持接入标准 MQTT Broker，设备状态实时上报，控制指令云端下发，适配工业物联网架构。

💡

光感自动控制

光敏电阻检测环境光照，自动调节 LED 照明，模拟化工场景中的光照闭环调节过程。

🌡️

温度监测与散热

DS18B20 数字温度传感器实时采集，超阈值自动启动继电器控制风扇散热。

🖥️

OLED 本地屏显

1.3 寸 SH1106 OLED 实时显示 WiFi 状态、系统模式、传感器数值与执行器状态。

🔌

双按键 + RGB 指示

总开关与辅助开关支持短按、长按组合操作；RGB LED 提供开机引导与状态指示。

操作说明

1
上电启动

接通 5V 电源，RGB LED 闪烁后常亮，OLED 显示系统信息

2
开启总开关

K1（按键 1）按下开启系统，进入自动模式；再次按关闭

3
切换模式

K2（按键 2）长按 2 秒切换自动/手动；短按循环切换 4 种手动状态

4
远程访问

浏览器输入设备 IP 打开控制面板，可远程监控与操作

5
MQTT 接入

通过串口配置 MQTT 服务器信息，设备自动上报状态至云端平台

典型教学案例

CASE 01

光照闭环控制实验

模拟化工车间照明自动调节场景。环境光照变化时，光敏电阻 ADC 值实时变化，系统根据设定阈值自动切换 LED 照明，演示化工控制中最基础的开关量闭环调节原理。

  • 理解 ADC 采样 — 数字转换链路
  • 掌握阈值比较的编程方法
  • 观察闭环反馈的执行效果
CASE 02

温度监测与散热保护

模拟化工反应釜超温保护场景。DS18B20 实时检测环境温度，超过 28°C 阈值自动启动继电器控制风扇散热，降至 25°C 自动停止，实现滞回比较控制。

  • 掌握 1-Wire 总线数字温度读取
  • 理解滞回比较器防止频繁切换
  • 学习继电器驱动大功率负载方法
CASE 03

物联网远程监控实训

模拟化工 DCS 集散控制架构。设备通过 WiFi 连接局域网，Web 界面实时显示传感器数据；同时通过 MQTT 与上层平台通信，实现设备状态远程感知与控制指令下发。

  • WiFi 配网与 Web 服务器搭建
  • MQTT 发布/订阅通信模型
  • JSON 数据序列化与解析
CASE 04

自动/手动模式切换实训

模拟化工控制系统的手/自动无扰切换。设备支持一键切换自动模式（传感器驱动）与手动模式（按键循环选择 4 种输出状态），帮助学生理解 DCS 系统操作方式。

  • 掌握状态机编程设计
  • 理解自动与手动模式逻辑
  • 熟悉按键去抖与长按短按识别
